# 🧮 Calculator Web Application

A simple and responsive **Calculator Web Application** built using **HTML, CSS, and JavaScript**. This project performs basic arithmetic operations with a clean, interactive, and user-friendly interface.

---

## 🚀 Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- 🧹 Clear (C) button
- ⌫ Backspace functionality
- 📱 Responsive design
- ⚡ Fast and lightweight
- 🎨 Attractive background with modern UI

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## ▶️ How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Double-click **calculator.html** or open it using **Live Server** in VS Code.
4. Start performing calculations.
